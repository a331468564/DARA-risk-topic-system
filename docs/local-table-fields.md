# 本地风险素材表字段

这份字段用于本地 CSV/XLSX 试跑，后续可以按同名字段迁移到飞书多维表。

| 字段名 | 类型建议 | 说明 |
| --- | --- | --- |
| 记录日期 | 日期 | 素材进入本地表的日期 |
| 来源 | 单选 | 36氪 / 巨潮资讯 / 华尔街见闻 / 格隆汇快讯 |
| 来源类型 | 单选 | official_rss / rsshub_public / cninfo_api |
| 素材层级 | 单选 | 线索池 / 证据池；媒体 RSS 默认线索池，巨潮公告默认证据池 |
| 原始标题 | 文本 | 新闻、快讯或公告标题 |
| 原文链接 | URL | 原始页面或 PDF 链接 |
| 发布时间 | 日期/文本 | 来源侧发布时间 |
| 原始摘要 | 长文本 | RSS 摘要、快讯正文或公告标题 |
| 涉及企业 | 文本 | 能确定企业主体时填写 |
| 涉及人物 | 文本 | 老板、实控人、法人、董事长等 |
| 风险类型 | 单选 | 债务风险 / 股权风险 / 合规风险 / 老板个人风险 / 供应链风险 / 舆情风险 / 融资风险 / 传承风险 |
| 风险信号 | 文本 | 命中的主要风险信号 |
| 风险关键词 | 文本 | 命中的关键词列表 |
| 老板视角 | 长文本 | 后续由 Agent 生成 |
| 可写选题 | 长文本 | 后续由 Agent 生成 |
| 短视频开头 | 长文本 | 后续由 Agent 生成 |
| 公众号标题 | 文本 | 后续由 Agent 生成 |
| 核验状态 | 单选 | 未核验 / 已核验 / 需进一步核验 / 放弃 |
| 处理状态 | 单选 | 待加工 / 已生成选题 / 已发布 / 已废弃 |
| 证据等级 | 单选 | A 官方公告/监管/交易所；B 公司官网；C 媒体报道；D 舆情线索 |
| 证据链接 | URL | 可用于核验的链接 |
| 去重键 | 文本 | 默认使用 URL 去重 |
| 备注 | 长文本 | 人工补充 |
